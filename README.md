## 🌟 چرا MOJA؟ (ویژگی‌های کلیدی)

- ⚡ **چند پروتکلی (Multi-Protocol):** نصب همزمان امن‌ترین پروتکل‌ها (Reality, WS, XHTTP, Hysteria2).
- 🛡️ **مسیریابی ضدسانسور (Anti-GFW):** 
  - مسدودسازی دسترسی به سایت‌های ایرانی از داخل VPN (برای جلوگیری از لو رفتن آی‌پی سرور).
  - بلاک کردن خروجی QUIC (UDP/443) برای دور زدن کندی یوتیوب و اینستاگرام در اپراتورهای ایرانی!
- 🌍 **خروجی WARP:** باز کردن بی‌دردسر سایت‌های تحریم‌کننده مثل ChatGPT، Spotify و Netflix.
- 📦 **سیستم مهاجرت جادویی (Export/Import):** بدون اینکه لینک ساب اسکریپشن توی گوشی اعضای خانوادتون رو عوض کنید، کل تنظیمات سرور قبلی رو با یک کلیک به سرور جدید منتقل کنید!
- 🎨 **داشبورد ساب‌اسکریپشن هوشمند:** لینک ساب شما در گوشی (V2rayNG) نقش کانفیگ رو بازی می‌کنه و توی مرورگر یک داشبورد گرافیکی شیک (Dark Mode) برای مشاهده حجم مصرفی باز می‌کنه!
- 🔐 **امنیت پیشرفته:** قابلیت اجبار به استفاده از `TLSv1.3` و تنظیم دستی `ALPN`.

## 🚀 روش اول: نصب پنل گرافیکی تحت وب (Web GUI) - 🔥 پیشنهاد ویژه

با این روش، شما فقط یک دستور می‌زنید و بعد از اون تمام تنظیمات (ساخت کانفیگ، روشن کردن WARP و...) رو از طریق یک وب‌سایت شیکِ نصب شده روی سرورتون مدیریت می‌کنید.

۱. با استفاده از SSH به سرور متصل بشید.
۲. دستور زیر رو کپی و اجرا کنید:
```bash
bash <(curl -fsSL https://raw.githubusercontent.com/gholamibromi/my-first-project/refs/heads/main/setupWEB.sh)
```
۳. اسکریپت از شما **پورت پنل**، **نام کاربری** و **رمز عبور** می‌خواد. اون‌ها رو وارد کنید.
۴. تمام! حالا آدرس `http://IP_Server:Port` رو تو مرورگر خودتون باز کنید، لاگین کنید و با موس کانفیگ‌هاتون رو بسازید!

## 💻 روش دوم: نصب نسخه ترمینالی (TUI)

اگر دوست دارید همه‌چیز رو از همون محیط سیاه ترمینال مدیریت کنید:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/gholamibromi/my-first-project/refs/heads/main/setupFINAL.sh)
```
بعد از اجرا، عدد `1` (New Installation) رو انتخاب کنید. سپس گزینه‌ی `Advanced Panel` رو بزنید تا به تمام تنظیمات حرفه‌ای دسترسی داشته باشید.

## 🔄 آموزش جابجایی سرور (بدون تغییر لینک توی گوشی کلاینت‌ها!)

سرورتون فیلتر شده؟ اصلاً جای نگرانی نیست!

### در نسخه گرافیکی وب (Web GUI):
1. تو سرور قدیمی، پنل گرافیکی رو باز کنید، کدی که تو قسمت Migration نوشته شده رو کپی کنید.
2. سرور جدید بخرید و پنل رو با دستور `setupweb.sh` روش نصب کنید.
3. کد رو توی بخش Import وارد کنید و دکمه Save & Rebuild رو بزنید!

### در نسخه ترمینال (TUI):
1. تو سرور قدیمی منو رو باز کنید و گزینه `6) Export/Import Migration` رو بزنید. کد رو کپی کنید.
2. تو سرور جدید اسکریپت `setupV2.sh` رو اجرا کنید، همون مسیر رو برید و Import رو بزنید.
3. در نهایت فقط کافیه آی‌پی سرور جدیدتون رو توی Cloudflare آپدیت کنید. لینکِ توی گوشی بدون یک ثانیه قطعی به کارش ادامه میده!

## ⚠️ نکات مهم و عیب‌یابی

- **باز بودن پورت‌ها:** مطمئن بشید پورت‌های پنل، Nginx (TCP)، Reality (TCP) و Hysteria2 (UDP) در فایروالِ پنل دیتاسنتر شما (مثلاً Hetzner یا DigitalOcean) باز باشن.
- **مشکل پینگ -1 در Hysteria2:** این مشکل معمولاً نشون‌دهنده‌ی بسته بودن پورت UDP تو فایروال شرکت ارائه‌دهنده سرورِ شماست.
- **حذف کامل:** برای پاک کردن پنل، کافیه توی اسکریپت ترمینال گزینه `Complete Removal` رو انتخاب کنید.
